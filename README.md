# Tycoons.com V1

This version keeps the main **Tycoons.com For Sale** header while adding a stronger V1 media concept.

Includes:
- Premium private-jet homepage
- Clear FOR SALE hero message
- Tycoons 100-style ranking cards
- Net worth estimates and source labels on the homepage
- Internal profile placeholder pages for every listed person
- Profile pages with net worth, source links, sector, region, companies, and rankings
- Names kept on one line where possible with improved card layout

Important:
Net worth estimates are volatile and should be reviewed before public launch.

Customize:
Replace `inquiries@tycoons.com` with your preferred inquiry address.
